# 💻 Sobre o Projeto do Agent 1 - Entrevista para Carreira em Tecnologia
---
Este projeto é uma ferramenta de orientação profissional focada em carreiras de tecnologia. Ele conduz o usuário por uma entrevista estruturada de 7 perguntas para identificar motivações, experiência, disponibilidade de estudo, preferências de trabalho e interesses técnicos. Com essas informações, o sistema analisa o perfil e recomenda as três carreiras mais compatíveis, destacando vantagens, desafios e contexto de mercado. O objetivo é ajudar iniciantes ou pessoas em transição a entenderem qual área de tecnologia combina melhor com seus objetivos e características, oferecendo clareza e direção antes de iniciar um plano de estudos.

## 📝 Perguntas Iniciais + Minhas Respostas

1. O que mais te atrai em tecnologia — resolver problemas, criar produtos ou entender sistemas?
   ➜ Resposta: resolver problemas

2. Você já tem experiência na área de tecnologia ou está começando do zero?
   ➜ Resposta: começando do zero

3. Quantas horas por semana você consegue dedicar aos estudos?
   ➜ Resposta: 14

4. No seu dia a dia, você prefere lidar mais com pessoas, dados ou código?
   ➜ Resposta: código

5. Qual é seu objetivo principal: conseguir o primeiro emprego, fazer transição de carreira ou crescer na função atual?
   ➜ Resposta: conseguir o primeiro emprego

6. Quais assuntos ou tecnologias mais despertam seu interesse?
   ➜ Resposta: desenvolvimento web, inteligência artificial, criação de aplicativos

7. Você tem alguma experiência prévia (mesmo que não seja em tech) que gostaria de aproveitar nessa nova jornada?
   ➜ Resposta: administrador e gestor de e-commerce

# 🚀 Projeto de Portfólio Principal

## **SmartVision – Classificador de Imagens com IA**

### O que Fazer
Criar um sistema completo que recebe uma imagem, classifica o conteúdo e retorna a previsão via API. O modelo deve ser treinado com um dataset público e disponibilizado online.

### Entregáveis
- Modelo treinado (CNN).
- API em FastAPI.
- Deploy em nuvem.
- Repositório GitHub com documentação.

### Critérios de Aceitação
- Acurácia mínima de 80%.
- API funcional e acessível.
- README com instruções claras.

### Dica
Use datasets como **CIFAR-10** ou **Cats vs Dogs** para acelerar o desenvolvimento.

# 💬 Roteiro de Entrevistas

### Pergunta 1: “Qual projeto de IA você desenvolveu?”
**Como responder:**  
Explique o problema → dados → modelo → métricas → resultado → o que aprendeu.

### Pergunta 2: “O que é overfitting?”
**Como responder:**  
“Quando o modelo aprende demais os dados de treino e perde capacidade de generalização.”

### Pergunta 3: “Como você escolhe um modelo?”
**Como responder:**  
“Depende do tipo de dado, objetivo e métricas desejadas.”

### Pergunta 4: “Explique a diferença entre regressão e classificação.”
**Como responder:**  
“Regressão prevê valores numéricos; classificação prevê categorias.”

### Pergunta 5: “Como você avalia um modelo?”
**Como responder:**  
“Métricas como acurácia, precisão, recall, F1-score e matriz de confusão.”
